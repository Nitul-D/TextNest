# TextNest

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Visit-blue)](https://Nitul-D.github.io/TextNest)
[![GitHub](https://img.shields.io/badge/GitHub-Repository-black)](https://github.com/Nitul-D/TextNest)

TextNest is a lightweight, web-based utility designed to help you manipulate and manage your text with ease. Whether you need to change text casing, remove extra spaces, or simply count your words, TextNest provides the tools you need in a clean, distraction-free environment.

## ✨ Features

- **Text Transformation**: Convert text to uppercase, lowercase, or reverse it
- **Copy & Paste**: Easily copy text to clipboard and paste from clipboard
- **Text Analysis**: Real-time word count, character count, space count, and reading time estimation
- **Dark Mode**: Toggle between light and dark themes for comfortable usage
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Privacy-Focused**: All processing happens in your browser - no data is sent to servers

## 🚀 Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Nitul-D/TextNest.git
cd TextNest
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

## 📖 Usage

1. **Enter Text**: Type or paste your text in the main textarea
2. **Transform Text**:
   - Click "Convert to Upper Case" to make all text uppercase
   - Click "Convert to Lower Case" to make all text lowercase
   - Click "Reverse Text" to reverse the order of characters
3. **Manage Text**:
   - "Copy All" copies the current text to your clipboard
   - "Paste" pastes text from your clipboard into the textarea
   - "Clear All" removes all text
4. **View Statistics**: See word count, character count, space count, and estimated reading time
5. **Toggle Theme**: Use the dark mode toggle in the navbar
